üìù Basic Usage

Using Direct AI Prompts

Call AI directly from any cell:

=AI_FILL("Summarize this text: {{1}}", A2)

Creating Custom Agents

  1. Open sidebar: BatchAI ‚Üí Start
  2. Click "Create Agent"
  3. Enter agent name (must be alphanumeric, no spaces)
  4. Write prompt template using {{1}}, {{2}} for parameters
  5. Select AI model (GPT-4 for complex tasks, GPT-3.5 for speed)
  6. Test your agent
  7. Click Save

Using Your Agents

=AI_FILL("@YourAgentName", A2, B2, C2)

Replace YourAgentName with your agent's name, and reference any cells you need.

Array Outputs

Control how results spread across cells:

  • Row output: =AI_FILL("List 3 fruits", "", "row")
  • Column output: =AI_FILL("List 3 points", "", "column")

‚ùì Common Issues & Solutions

üî¥ "API key not configured" error

Solution: Go to BatchAI ‚Üí Start ‚Üí Settings and add your OpenAI API key. You can get one at platform.openai.com/api-keys.

üî¥ "Agent not found" error

Solution:

  • Make sure agent name starts with @ symbol in formulas
  • Check spelling (agent names are case-sensitive)
  • Verify the agent was saved successfully in the sidebar

üî¥ Formula returns "#ERROR!"

Common causes:

  • Missing parameters: Check if your agent expects parameters (e.g., {{1}}) and you're providing them
  • Invalid API key: Verify your API key is correct
  • Syntax error: Make sure formula syntax is correct: =AI_FILL("prompt", reference)

üî¥ Slow performance

Solutions:

  • Use GPT-3.5 instead of GPT-4 for faster responses
  • Process rows in smaller batches
  • Simplify your prompts
  • Check your internet connection

üî¥ "Quota exceeded" error

Solution: This means your OpenAI account has run out of credits. Check your usage and billing at OpenAI's billing page.

üî¥ Sidebar not opening

Solution:

  • Refresh the page (F5 or Cmd+R)
  • Check if you have pop-ups blocked
  • Try a different browser (Chrome recommended)
  • Clear browser cache

üß™ Advanced Features

Multiple Parameters

Drag Down Formulas

Process entire columns by dragging formulas down like regular Excel functions!

  1. Write your formula in the first row
  2. Click the cell
  3. Drag the blue handle down
  4. All rows process automatically!

Web Search Tools

Enable web search for agents that need real-time information:

  1. When creating an agent, check the "Web Search" tool
  2. Your agent can now search the internet for current data
