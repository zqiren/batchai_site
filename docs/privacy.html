Privacy Policy

Last updated: December 29, 2024

1. Information We Collect

BatchAI operates as a Google Sheets add-on. We collect and process:

  • User-Provided Data: API keys for AI providers (stored locally in Google Apps Script properties)
  • Spreadsheet Content: Data you choose to process with AI formulas (processed in real-time, not stored)
  • Usage Data: Basic analytics on feature usage to improve the service

2. How We Use Your Information

  • To provide AI processing services via third-party APIs (OpenAI, Anthropic, etc.)
  • To store your custom AI agent configurations
  • To improve product features and user experience

3. Data Storage and Security

  • API Keys: Encrypted and stored in Google's secure PropertiesService
  • Agent Configurations: Stored in your Google account
  • Spreadsheet Data: Never stored on our servers; processed in real-time only

4. Third-Party Services

BatchAI integrates with AI providers. When you use AI features, your prompts are sent to:

  • OpenAI (if you configure OpenAI API key)
  • Anthropic Claude (if you configure Anthropic API key)
  • DeepSeek (if you configure DeepSeek API key)
  • Google Gemini (if you configure Google API key)

Each provider has their own privacy policies. We recommend reviewing them.

5. Data Retention

We retain your data only as long as your Google account is active. You can delete:

  • API keys anytime via Settings
  • Custom agents anytime via the sidebar
  • All data by uninstalling the add-on

6. Your Rights

You have the right to:

  • Access your data (view in Settings/Agent tabs)
  • Delete your data (remove API keys, delete agents)
  • Export your data (contact us for assistance)

7. Children's Privacy

BatchAI is not intended for users under 18. We do not knowingly collect data from children.

8. Changes to Privacy Policy

We may update this policy. Changes will be posted on this page with updated date.
